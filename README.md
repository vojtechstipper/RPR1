### Docker
V hlavní složce se nachází soubor compose.yml. V této složce je potřeba otevřít terminál a vložit příkaz:
```bash
docker compose up
```
Poté se vytvoří kontejner s Microsoft SQL Server, který je potřebný pro běh BE.
